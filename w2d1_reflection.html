Useful Tips

  • Use to_json || to_yaml and JSON::parse(some_json) || YAML::load(some_yaml) to convert Ruby objects into the respective formats (Note: Unlike YAML, JSON does not natively support converting custom classes, although it is possible to overwrite the to_json method)
  • In pry, you can reference the previous return value with an underscore (particularly useful when you forget to save something as a variable)
