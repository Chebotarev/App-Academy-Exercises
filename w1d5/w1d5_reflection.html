It was another tough but enlightening day at App Academy. We finished off the first week with a lecture on basic data structures given by Ryan Glassett. He gave an overview of linked lists, stacks and queues, as well as the methods the Ruby Array class has to replicate each structure. We looked at trees and how to implement breadth first search and depth first search.

There were three major projects today:

  • PolyTreeNode - A class simulating a tree that can have many children
  • Knight's Travail - Finding the shortest path a knight can take from one position on a chess board to another utilizing the PolyTreeNode class
  • TicTacToe Artificial Intelligence - An AI for the Tic Tac Toe game we implemented earlier in the week that is unbeatable (assuming you don't cheat)
The hardest of these problems was the Tic Tac Toe AI. It works by traversing a tree representing all possible states of the board and only choosing moves that will lead to a winning state (or at the very least not a losing state)
